# Accessibility Navigation Standards

1. Below is a description of what actions a user can implement when using a specific key on a keyboard. 
2. If the user is using a single-button switch, it acts as an Enter button.

## Tab
Move the focus indicator forward

## Shift+Tab
Move the focus indicator backward

## Enter
1. Activate a link
2. Activate an action button
3. Select an item from a menu

## Space
1. Check/uncheck a checkbox
2. Select/deselect a radio button
3. Open/close a drop-down menu
4. Navigate a page downward if scrolling down is possible

## Shift+Space
Navigate a page upward if scrolling up is possible

## Arrow Keys
1. Navigate through radio buttons
2. Navigate through menu items
3. Move a cursor directionally
4. Increase/decrease slider values

## Escape
Close modal or dialog boxes

NOT USED IN PURRAMID APPLICATIONS AT THIS TIME:
## Shift +/-
Zoom in/out (this is different than resizing a window)
